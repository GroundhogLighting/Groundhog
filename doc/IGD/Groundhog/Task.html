Overview

This class is the parent of all the pre-defined tasks (i.e. calculate DC matrix). it contains basic methods that are common for every task.

Every task contains a target, a proc and dependencies.

The target is what makes it unique when compared to other similar tasks (i.e. calculating the luminaire contribution for one workplane is different from calculating the luminaire contribution to another workplane).

The proc is a Proc object that is run when solved. It is intended to return the necessary commands that, when run, will accomplish such task (i.e. rfluxmtx -ab …)

The dependencies is an array with other Tasks that must be run before this task

def expand
    if !@dependencies or @dependencies.length == 0 then
        return [self]
    else
        ret = [self]
        @dependencies.each {|dep|
            ret = ret + dep.expand
        }
        return ret
    end
end

def solve(options)
    success = @proc.call(options)
    warn "Fatal: Impossible to solve #{self}" if not success
    return false if not success
    return success
end
