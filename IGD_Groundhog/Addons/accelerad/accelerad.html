Accelerad add-on

This add-on allows using Accelerad, which is a GPU accelerated Physically Based ray-tracer inspired in Radiance. If enabled and properly configured, it may lead to major speed improvements in some calculations.

More info

The Groundhog team has developed this add-on in order to allow this extension to benefit from Accelerad. However, Accelerad is not distributed along with Groundhog, so please refer to the authors website for more information.

Accelerad is being developed by Nathaniel Jones at the MIT Sustainable Design Lab.
